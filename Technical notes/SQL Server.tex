\section{Find the name of the instance}
\begin{itemize}
    \item Open Start and type "SQL Server Configuration Manager".
    \item Go to **SQL Server Services**.
    \item Look for SQL Server (SQLEXPRESS) or another instance name.
\end{itemize}
\section{Find the connection string}
\begin{itemize}
    \item The first time is the hardest since you should have copied it from setup wizard output
    \item You can build it using the connect to database engine window.
\end{itemize}
